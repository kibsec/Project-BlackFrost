Project BlackFrost
=================

Description:
------------
Project BlackFrost is a Python-based application implementing a **cascade encryption system**. 
The source code is fully compiled for security; no Python `.py` files are included. 

Author:
-------
Gerard_Cea

Project:
--------
Xaltius Academy

Binaries:
---------
- Project_BlackFrost_Linux.bin    -> Run on Linux (chmod +x before execution)
- Project_BlackFrost_Windows.exe  -> Run on Windows

Usage:
------
Linux:
  $ chmod +x Project_BlackFrost_Linux.bin
  $ ./Project_BlackFrost_Linux.bin

Windows:
  Double-click Project_BlackFrost_Windows.exe or run via Command Prompt.

Assets:
-------
Any additional resources required by the application are stored in the assets/ folder.

Security:
---------
- Implements cascade encryption for enhanced security.
- Source code is fully compiled; no .py files are exposed.
- All dependencies are included within the binaries.

GitHub Deployment:
-----------------
- Only binaries, assets, and this README are included.
- Python source, virtual environments, and build artifacts are excluded via .gitignore.
